ЛОКАЛЬНО (одна машина):
  1. cd PiServer && dotnet run
  2. Откройте браузер: http://localhost:5000
  3. cd PiClient && dotnet run
  4. Введите URL: http://localhost:5000

РАСПРЕДЕЛЁННО (несколько машин):
  1. На машине A: cd PiServer && dotnet run
  2. На машине B: cd PiClient && dotnet run
     Введите: http://<IP_машины_A>:5000
  3. Откройте браузер: http://<IP_машины_A>:5000
