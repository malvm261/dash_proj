## 🚀 Как запустить

### Вариант 1: Один компьютер (локальный)
```bash
# Терминал 1
cd PiServer && dotnet run

# Браузер
http://localhost:5000

# Терминал 2
cd PiClient && dotnet run
```

### Вариант 2: Несколько компьютеров (распределённо)
```bash
# Машина A (сервер)
cd PiServer && dotnet run

# Машина B (клиент 1)
cd PiClient && dotnet run
# Введите: http://192.168.1.A:5000

# Машина C (клиент 2)
cd PiClient && dotnet run
# Введите: http://192.168.1.A:5000

# Браузер на любой машине
http://192.168.1.A:5000
```

### Вариант 3: Исполняемые файлы
```bash
# Скопируйте exe файлы:
PiServer.exe
PiClient.exe

# Запустите:
PiServer.exe    # на сервере
PiClient.exe    # на клиентах
```
