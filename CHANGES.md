# Изменения для добавления веб-интерфейса

## 📋 Что было добавлено

### 1. Веб-интерфейс (wwwroot/)

Создана папка `PiServer/wwwroot/` с тремя файлами:

#### ✨ `index.html` (2.9 KB)
- Дашборд в браузере
- Статистика в реал-тайме (прогресс, π значения)
- График сходимости π к эталонному значению
- Список активных клиентов
- Лог событий сервера
- Управление параметрами вычислений

#### 🎨 `style.css` (5.5 KB)
- Современный дизайн (gradient background)
- Адаптивный layout (работает на мобильных)
- Тёмное оформление для логов
- Smooth анимации и переходы

#### 📡 `client.js` (9.5 KB)
- Chart.js для отрисовки графика π
- Polling API каждые 1 сек
- Обновление UI в реал-тайме
- Список активных клиентов
- Управление вычислениями (reset параметров)

### 2. Изменения сервера (PiServer/Program.cs)

```csharp
// Добавлены:
- app.UseStaticFiles();           // отдача wwwroot/
- app.UseCors();                  // разрешить cross-origin запросы
- GET /                           // отдача index.html
- GET /api/clients                // список активных клиентов
- Отслеживание активных клиентов в Coordinator

// В Coordinator добавлены:
- Dictionary<string, (int, double)> _clients  // какие клиенты считают
- GetActiveClients()              // вернуть список клиентов
- Счётчик блоков для каждого клиента
```

## 🚀 Как использовать

### 1. Запустить сервер
```bash
cd PiServer
dotnet run
```
Консоль:

## 🔧 Технические детали

### Как работает polling в браузере
```javascript
// Каждые 1 сек браузер запрашивает:
GET /api/status    → получить текущий статус
GET /api/clients   → получить список активных клиентов

// Данные отображаются на графике
chart.data.datasets[0].data.push(pi1_value);
chart.update();  // перерисовка графика
```

### Какие файлы из wwwroot/ отдаются
```
GET /            → index.html
GET /style.css   → style.css
GET /client.js   → client.js
GET /api/status  → JSON (API)
```
